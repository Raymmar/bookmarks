Universal Bookmarks - Spec & Requirements

Product Overview

Universal Bookmarks is a web and browser-extension-based system that enables users to collect, organize, search, and visually explore bookmarks and web content. It serves as a visual memory system and research assistant, using AI to extract insights, tag content, and build a searchable, interactive graph of the user's web exploration.

Core Features

Web App

Manual URL Input: Users can paste URLs directly.

Content Unfurling: Extracts metadata, summary, and key content.

Auto-tagging & Vector Embedding: For intelligent filtering and search.

Browser Extension

Capture Page Info:

URL

User notes (multiple per page)

Screenshots (upload & gallery support)

DOM selections (HTML snippets)

Text highlights with in-page indicators

User Tags: Manual tagging

System Tags: Auto-generated for analysis

Data Ingestion Sources

Browser Extension

Manual Web Entry

Third-party integrations (Twitter, Pinterest, Chrome Bookmarks, etc.)

Visual & Interactive Features

Force-directed graph view to explore bookmark clusters

Visual canvas where nodes attract related content

Sidebar with bookmark details, highlights, and notes

Search bar with tag, topic, and recency filters

Chat UI to query bookmarks and get AI responses

IDE-style activity feed for browsing + curation timeline

AI Functionality

RAG-style search through all stored content

Chat-based interface for querying & summarization

Contextual search window adjustable by tags, dates, sources, etc.

Insight Depth Tiers:

L1: On-page content only

L2: One-click away content

L3: Multi-layered link graph

L4: Full research sweep (news, social, background)
